# HausTap Web App (PHP)

This is the main PHP web application.

## Run Locally

- `php -S localhost:8002 router.php`
- Open `http://localhost:8002/`

## Dependencies

- `PHP 8.1+`
- Optionally reference shared libraries from `../../backend/vendor/autoload.php` if used.

## Environment

- Place any app-specific configuration in `.env` files at repo root if needed.
- No UI/UX changes are introduced by this document.
