# HausTap Admin Panel (PHP)

This directory hosts the admin panel.

## Run Locally

- `php -S localhost:8003 -t admin_haustap/admin_haustap`
- Open `http://localhost:8003/dashboard.php`

## Notes

- Requires `PHP 8.1+`.
- Adjust paths and includes as needed for your environment.
- This document does not change UI or UX.
